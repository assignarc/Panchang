
**Project Objectives**
1. Create a library that can compute Panchag and other Vedic astrology related calculations.
1. Integrate with Swiss Ephemeris for accurate planetary positions.
1. Integrate with SwissEphNet for cross-platform compatibility.
1. Create test suit to ensure accuracy and reliability of calculations.
1. Keep user interface available from the start, but not the main focus of the project.
